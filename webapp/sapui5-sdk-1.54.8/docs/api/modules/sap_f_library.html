Module sap/f/library

Contained Namespaces, Types, Enums & Classes
sap.f SAPUI5 library with controls specialized for SAP Fiori apps.
sap.f.AvatarImageFitType Types of image size and position that determine how an image fits in the sap.f.Avatar control area.
sap.f.AvatarShape Types of shape for the sap.f.Avatar control.
sap.f.AvatarSize Predefined sizes for the sap.f.Avatar control.
sap.f.AvatarType Types of sap.f.Avatar based on the displayed content.
sap.f.DynamicPageTitleArea Defines the areas within the sap.f.DynamicPageTitle.
sap.f.DynamicPageTitleShrinkRatio A string type that represents the shrink ratios of the areas within the sap.f.DynamicPageTitle.
sap.f.LayoutType Layouts, representing the number of columns to be displayed and their relative widths for a sap.f.FlexibleColumnLayout control.
